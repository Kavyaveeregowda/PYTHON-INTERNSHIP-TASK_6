# Task 6 – Flask Portfolio Website

This project is created as part of **Python Internship – Task 6**.  
The goal is to build a simple **Portfolio Website** using **Flask**, HTML, and CSS.

## ?? Project Overview
The website includes:
- Profile photo  
- Name, role, and about section  
- Skills  
- Projects  
- Contact form  
- Resume download button  
- CSS styling using static files  

## ?? Folder Structure
```
app.py
templates/
    index.html
    thankyou.html
static/
    style.css
    profile.jpg
    resume.pdf
```

## ?? How to Run
1. Install Flask  
   `pip install flask`

2. Run the app  
   `python app.py`

3. Open in browser  
   `http://127.0.0.1:5000/`
